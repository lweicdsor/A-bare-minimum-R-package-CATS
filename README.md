# A-bare-minimum-R-package-CATS
The R package cats. This package is to show how a R package look like. You can run the function cats_function() to start.

# Installation
```java
library(devtools)
install_github("lweicdsor/A-bare-minimum-R-package-CATS")
```

library(cats)

cats_function()
